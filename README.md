# Machine Learning Labs

A collection of Machine Learning projects implemented using Python, Pandas, Scikit-Learn, Matplotlib, and Seaborn.

## Projects

| # | Project | Model(s) Used | Best Accuracy |
|---|---------|---------------|---------------|
| 1 | Student Performance Prediction | Decision Tree | 93% |
| 2 | Customer Churn Prediction | Decision Tree, Random Forest | 79% |
| 3 | Heart Disease Prediction | Logistic Regression, Random Forest | 98.5% |
| 4 | Credit Card Fraud Detection | Random Forest | 99% |
