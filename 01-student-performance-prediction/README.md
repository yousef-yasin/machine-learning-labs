# Student Performance Prediction

## Overview

This project predicts student academic performance using Machine Learning.

The model analyzes factors such as:

- Study Hours
- Attendance
- Sleep Hours
- Internet Access
- Previous Grades
- Family Support

and predicts the student's final performance.

## Objectives

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Performance Prediction

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
